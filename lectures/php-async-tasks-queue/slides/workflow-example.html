Example

  1. User orders food and should get email notification about order
  2. Application handles request, save data and enqueue email job
  3. Application responds to the user with success message
    1. Job manager looks to the job queue
    2. Job manager finds new task in queue and picks it out
    3. Job manager ask job worker to do this task
    4. Job worker does job and tell about it to the manager
    5. Email is sent!
  4. Everyone is happy!
