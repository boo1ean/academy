##  Pseudo-classes

```css
/* unvisited links */
a:link {}

/* visited links */
a:visited {}

/* links on mouse over */
a:hover {}

/* input element which has focus */
input:focus {}

/* <p> elements that is the first child of its parent */
ul:first-child {}

/* first letter of every <p> element */
p:first-letter {}
```
