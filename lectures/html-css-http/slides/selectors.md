### CSS

```css
/* Universal selector */
* { color: red; }

/* Type selectors */
div { padding: 100px;  }

/* Descendant selectors */
div a { text-decoration: underline; }

/* Child selectors */
div > a { text-decoration: underline; }

/* Id selector */
#content { margin-left: 20px; }

/* Class selector */
.btn { background: url("img/btn.png"); }
```
