## Selectors

```css
/* Universal selector */
* { color: red; }

/* Id selector */
#content { margin-left: 20px; }

/* Class selector */
.btn { background: url(/img/btn.png); }

/* Type selectors */
div { padding: 100px;  }

/* Descendant selectors */
li p { text-decoration: underline; }

/* Child selectors */
li > p { text-decoration: underline; }
```
